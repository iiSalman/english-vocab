# English vocabulary library

Advanced (C1/C2) English words mined from TV show dialogue, for an IELTS-7 learner.

Each season/episode page has:
- A grid of vocabulary cards with definitions, examples from the show, and AI-generated illustrations
- A short NotebookLM audio podcast covering every word

Open `index.html` to browse.
